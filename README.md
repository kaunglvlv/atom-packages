Atom Packages
=============

These are the packages for Atom I can't live without.

Requirements
------------

- Atom
- Git

Installation
------------

Clone this repo

```
$ git clone https://github.com/kkaung/atom-packages
```

Run apm install

```
$ apm install --packages-file packages.txt
```

Updating
--------
```
$ apm list --installed --bare > packages.txt
```
